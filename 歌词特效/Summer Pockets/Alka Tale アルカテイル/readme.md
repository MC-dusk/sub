# 歌词

`Alka Tale`原为`Summer Pockets`游戏op，本字幕用于`Summer Pockets REFLECTION BLUE` Pocket线ed2

> **アルカテイル**
>
> **Alka Tale**

- [x] 精准逐字
- [ ] 平均逐字
- [ ] 逐行

- `歌词特效`中日双语，提供模板行（模板由[星空字幕组](https://www.bilibili.com/video/BV1o64y1W7xg)制作），需自行自动化生成输出字幕

- `普通字幕`中日双语+罗马音，`\kf`卡拉OK效果

**二者均为纵向字幕**
